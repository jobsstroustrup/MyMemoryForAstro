# 气候物理与大气科学 (Climate Physics & Atmospheric Science)

## 中文版

### 熟练度
精通（学术研究级别）

### 描述
- 对流动力学：烟羽模型（MTT）、LES、对流层顶约束
- 大气辐射：辐射加热率、辐射平衡分析
- 大尺度环流：Walker 环流、Hadley 环流及其对全球变暖的响应
- 核冬天假说：平流层烟尘注入机制、烟尘寿命与辐射效应
- 气溶胶物理：消光系数、AOD、反演算法
- 甲烷遥感：TROPOMI 数据处理、排放源归因

### 🌟 研究级 signature capabilities

- **Linear Superposition framework (Zhenyu 原创, 2021)** — Walker Circulation 对 oceanic forcing 宽度响应的 2nd-order Taylor expansion framework, 4-field validation SST / Precip / ω / SW correlation 0.83-0.93 (见 [[Linear-Superposition-Framework]])
- **Nuclear Winter 5-phase physical understanding** — 1-D thermo (T,q vs h,q 3.58e-5 J/kg validation) → MTT momentum (Hiroshima L_w≈9.5km LNB≈6.6km vs 15.7km photo) → DAM 3-D LES entrainment (DNS 3-30% match) → 46k core-hours production (CERA20C 534 ton soot) → DSE+SWTG methodology workshop ([[Nuclear-Winter-Phase-4p5-Methodology-Workshop]])
- **Methane 3-scale analysis** — TROPOMI aggregated precision 9.76e-5° ≈ 10.87m + density+quantile regression reformulation + negative-slope→positive-slope diagnosis (Nebraska R²=0.633)
- **MODIS+CALIPSO joint retrieval algorithm** — First iteratively-coupled (本科毕业论文, 4 Beijing case study abs AOD↓ 0.011-0.032)
